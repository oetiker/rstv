# TLabel  (guide pp. 465–467)

Rust module(s): src/widgets/static_text.rs   |   magiblot: include/tvision/dialogs.h / source/tvision/tlabel.cpp

> TLabel has two own fields (`Light`, `Link`), four methods documented by the guide
> (`Init`, `Draw`, `GetPalette`, `HandleEvent`), plus `Load`/`Store`/`shutDown`
> lifecycle methods visible in the header. The magiblot C++ source also exposes
> `shutDown` and the private helper `focusLink`.
